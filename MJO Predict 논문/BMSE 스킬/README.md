# 전체적으로, BMSEP는 검토된 40일 기간 동안 BMSEa보다 크며, MJO 위상 오류가 전체 오류를 지배한다는 것을 암시한다.

# 이러한 결과는 BMSEp가 BMSE와 MJO 예측 기술의 성장에 BMSEa보다 더 중요한 역할을 한다는 것을 암시한다.
