# 1) 주어 be pp 목적어 => 주 동 간목 직목이 변한것 간목이 주로 수동태의 주어가 된다.
## we are taught French by mrs.moor on Tuseday
## mrs.moor teach us French on Tuseday
## taught 탓, 톳
# 일반적으로 간목이 수동태의 주어로 더 많이 쓰임
